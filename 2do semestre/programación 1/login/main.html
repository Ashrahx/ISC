PHP is an open source programming language that is mainly used to create dynamic web applications.

Among the PHP characteristics are included:

  • Syntax similar to C
  • Compatibility with a wide variety of databases
  • Object -oriented programming support
  • wideDocumentationAndDeveloperCommunity

PHP is executed on the server side, which means that the PHP code is processed on the web server before the user's browser is sent.This allows to create dynamic web pages that can show personalized content based on user applications.
